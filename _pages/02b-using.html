The framework is based on 30 areas, each containing 3 levels:

  • Level 1: Foundation
  • Level 2: Practitioner
  • Level 3: Advanced

The levels are an indication of the amount of responsibility a person may have in this area.

Feel free to use it for non-commercial purposes.

To incorporate the competencies into your project, you can add them as a git submodule:

git submodule add https://github.com/Wyver-Solutions/_elearning-competencies _elearning-competencies

The underscore is so they work directly as a Jekyll collection. You will need to run the above command inside your Collections directory, and then use git pull inside the submodule whenever you want to update.
